В этом проекте есть статическая библеотека, а для создания динамической есть Makefile.
Не уверен, что Makefile будет работать где-то кроме терминала Git bash, так как создавал я его для использования в Git bash.
В самом Makefile есть 4 функции:
  1. all - создана для удаления динамической библеотеки(DinMylib.dll) и программы запуска(programm.exe),
  2. build - для создания динамической библеотеки и программы запуска,
  3. clean - для удаления, используется в all вместе с функцией build,
  4. run - сам запуск программы в терменале.
  
Для использования функций в терминали у вас должен быть установлен сам "make", можете попробовать использовать их через "make all(название нужной функции)",
возможно, через make ничего не будет работать, тогда попробуйте "mingw32-make all", всё зависит от скаченного вами компилятора.
В общем и целом это всё, надеюсь что у других это будет работать, впринципе можно попробовать написать Makefile для VS code сильно он отличаться не будет.
