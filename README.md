В этом проекте есть статическая библиотека, а для создания динамической есть Makefile. Не уверен, что Makefile будет работать где-то кроме терминала Git Bash, так как создавал я его для использования именно в Git Bash.

В самом Makefile есть 4 цели (функции):

all — для удаления динамической библиотеки (DinMylib.dll) и программы запуска (programm.exe), а затем их пересборки

build — для создания динамической библиотеки и программы запуска (без удаления)

clean — для удаления файлов (используется в all вместе со сборкой)

run — для запуска программы в терминале

Для использования целей в терминале у вас должен быть установлен сам make.
Можете попробовать использовать их через make all (или make + название нужной цели).
Если через make ничего не будет работать, попробуйте mingw32-make all — всё зависит от того, какой компилятор вы скачали.

Важно: сначала используйте цель build, а уже после — run.

В общем и целом это всё. Надеюсь, что у других это будет работать. В принципе, можно попробовать написать Makefile для VS Code — сильно он отличаться не будет.
